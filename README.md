# Test Rig GUI Application

Qt Quick-Based Styling + Widgets to assist test rig ops.

## Filesystem Structure
- `qml`: Qt Quick
- `src`: C++ Sources and Includes

To start building, run this in the root directory of this repo:
```sh
cmake -S . -B build
```
then
```sh
cmake --build build
```
for example.
